##### Steps to get the projects in the local machine #####
1. Go to the repository page on GitHub. - https://github.com/dipesh31/ipl-leaderboard
2. Click the green <> Code button.
3. Copy the HTTPS link (it should look like https://github.com/dipesh31/ipl-leaderboard.git)
4. In Eclipse, go to File > Import...
5. Expand the Git folder and select Projects from Git (with smart import). Click Next.
6. Select Clone URI and click Next.
7. Paste the URL
8. Click Next, select the branches you want (usually main), and click Next again.
9. Local Destination: Choose where you want the code stored on your hard drive. Click Next.
10. Eclipse will scan the folder for projects (like Maven, Gradle, or standard Java). Click Finish.
